## Instagram Basic Display API Flutter

-EN-

You can use the Instagram Basic Display API to allow your users to showcase the photos, videos and albums they share on Instagram.

In this project, I wanted to show how we can make requests to the API I mentioned.

If you want to clone and use the application, you can email me and I will send you an invitation to add your Instagram account to my test users, you can use it after you accept this request. I plan to publish a Medium article on this subject in the future.
